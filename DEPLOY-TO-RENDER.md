## ğŸ› Troubleshooting

### **Build Fails**

**Error: "Module not found"**
```bash
# Make sure package.json is correct
# Check build logs for specific missing package
# Add missing dependency:
npm install missing-package --save
git commit -am "Add missing dependency"
git push
```

**Error: "Build timeout"**
- Free tier has 10-minute build limit
- Optimize by removing unused dependencies
- Consider upgrading plan

### **Site is Slow / Spins Down**

This is normal on free tier. Solutions:
1. **Accept it** - Fine for landing page
2. **Use a pinger** - Service to ping your site every 10 min (keeps it awake)
3. **Upgrade to paid tier** ($7/month) - No spin-down

### **Database Connection Failed**

1. Check `DATABASE_URL` is set correctly
2. Verify database is in same region as web service
3. Use **Internal Database URL** (not external)
4. Check database is running (Render dashboard)

### **Environment Variables Not Working**

1. Check spelling (case-sensitive!)
2. Click "Save Changes" after adding variables
3. Wait for auto-redeploy to complete
4. Check logs for errors

### **Custom Domain Not Working**

1. Wait 15 minutes for DNS propagation
2. Check DNS records at https://dnschecker.org
3. Verify CNAME points to: `creai.onrender.com`
4. Clear browser cache
5. Try incognito mode
